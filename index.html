JOXIDE

JOXIDE - компилятор java с более гибкой системой типов, минимальным количеством шаблонного кода и современными идеями.

Начать 
public class Main {

    static String greet(String text) {
        return "Hello " + text;
    }

    public static void main(String[] args) {
        val expr = "Artyom" |> greet();
        System.out.println(expr);
    }

}
